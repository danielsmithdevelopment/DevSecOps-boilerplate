# DevSecOps-boilerplate

Boilerplate automation for DevSecOps: golden images (Packer), configuration management (Ansible), infrastructure as code (Terraform, Pulumi), observability (Grafana LGTM+), runtime security, and GitHub Actions CI/CD with supply-chain gates.

**Documentation:** [docs/README.md](docs/README.md) — one page per tool with quick start, configuration, and official resource links.
